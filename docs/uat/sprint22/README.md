# Sprint 22 UAT Evidence Folder

This folder contains the Sprint 22 UAT pilot pack for Warehouse, Sales, and QC.

Commit templates and sanitized evidence only. Do not commit raw screenshots, exports, logs, passwords, tokens, customer PII, private addresses, phone numbers, or commercial secrets.

## Structure

```text
templates/      CSV and report templates used during UAT
evidence/       Sanitized screenshots, logs, exports, and session notes
```

## Required Templates

```text
templates/uat_users_roles.csv
templates/seed_data_plan.csv
templates/scenario_results.csv
templates/observation_log.csv
templates/issue_triage_board.csv
templates/session_schedule.csv
templates/go_no_go_report.md
```

## Evidence Handling

Raw UAT evidence should be reviewed and redacted before it is committed.

Use these subfolders for sanitized evidence:

```text
evidence/screenshots
evidence/logs
evidence/exports
evidence/session-notes
```
