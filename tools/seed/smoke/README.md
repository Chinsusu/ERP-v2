# Smoke Seed Data

`sprint0_smoke_seed.json` is the Sprint 0 smoke data contract for QA and automation.

It is intentionally small and deterministic:

- one admin login,
- one warehouse lead and one warehouse staff user,
- master data shells for SKU, batch, supplier, factory, and customer,
- one stock adjustment movement,
- one shipping manifest with scan codes for success and negative smoke.
